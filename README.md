## Per_fl-and-FedAvg
PyTorch implementation of Per-FedAvg (Personalized Federated Learning: A Meta-Learning Approach) \
and FedAvg(Communication-Efficient Learning of Deep Networks from Decentralized Data).

## Environment
numpy == 1.20.2
pytorch == 1.4.0
tqdm == 4.55.1

## Usage
# 1.build data 
'''
python data_build
'''
# 2.start training
'''
python main.py
'''
